Introducción a JavaScript: Variables y Funciones

Variables en JavaScript

Las variables son contenedores para almacenar datos en JavaScript.

Declaración de variables:

  • var: Declaración de variable con ámbito de función (obsoleto en código moderno)
  • let: Declaración de variable con ámbito de bloque (recomendado)
  • const: Declaración de constante con ámbito de bloque

Ejemplos:

// Usando let
let nombre = "Juan";
let edad = 25;
let esEstudiante = true;

// Usando const para valores que no cambiarán
const PI = 3.14159;
const DIAS_SEMANA = 7;

// Reasignación (solo posible con let)
nombre = "María";
edad = 26;

console.log(nombre); // Salida: María
console.log(edad);   // Salida: 26

Recuerda: usa const por defecto y let cuando necesites reasignar valores.

Tipos de Datos en JavaScript

JavaScript es un lenguaje de tipado dinámico, lo que significa que las variables pueden contener diferentes tipos de datos.

Tipos de datos primitivos:

  • String: Cadenas de texto
  • Number: Números (enteros y decimales)
  • Boolean: true o false
  • Undefined: Variable declarada pero sin valor asignado
  • Null: Valor nulo (ausencia intencional de cualquier valor)
  • Symbol: Identificador único (ES6+)

Ejemplos:

let nombre = "Ana";         // String
let edad = 30;              // Number
let altura = 1.75;          // Number (decimal)
let esEstudiante = false;   // Boolean
let direccion;              // Undefined
let valorNulo = null;       // Null

console.log(typeof nombre);        // Salida: string
console.log(typeof edad);          // Salida: number
console.log(typeof esEstudiante);  // Salida: boolean
console.log(typeof direccion);     // Salida: undefined
console.log(typeof valorNulo);     // Salida: object (esto es un error conocido en JS)

El operador typeof se usa para determinar el tipo de una variable.

Operadores en JavaScript

Los operadores permiten realizar operaciones en variables y valores.

Tipos de operadores:

  • Aritméticos: +, -, *, /, %, **
  • Asignación: =, +=, -=, *=, /=
  • Comparación: ==, ===, !=, !==, >, <, >=, <=
  • Lógicos: &&, ||, !

Ejemplos:

// Operadores aritméticos
let suma = 5 + 3;       // 8
let potencia = 2 ** 3;  // 8 (2 elevado a 3)

// Operadores de asignación
let x = 10;
x += 5;  // Equivale a: x = x + 5;

// Operadores de comparación
console.log(5 == "5");   // true (compara solo valor)
console.log(5 === "5");  // false (compara valor y tipo)

// Operadores lógicos
let a = true, b = false;
console.log(a && b);  // false
console.log(a || b);  // true
console.log(!a);      // false

Los operadores son fundamentales para realizar cálculos y tomar decisiones en tu código.

Funciones en JavaScript

Las funciones son bloques de código reutilizables que realizan una tarea específica.

Declaración de funciones:

// Función declarativa
function saludar(nombre) {
    return `Hola, ${nombre}!`;
}

// Función expresiva
const despedir = function(nombre) {
    return `Adiós, ${nombre}!`;
};

// Función flecha (arrow function)
const multiplicar = (a, b) => a * b;

Uso de funciones:

console.log(saludar("María"));     // Salida: Hola, María!
console.log(despedir("Juan"));     // Salida: Adiós, Juan!
console.log(multiplicar(4, 5));    // Salida: 20

Parámetros y retorno:

  • Los parámetros son valores que la función recibe
  • El retorno es el valor que la función devuelve (usando return)
  • Si no se especifica un retorno, la función devuelve undefined

Las funciones son esenciales para organizar y reutilizar código en JavaScript.
